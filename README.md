# taskpalvi
Tarea Palvi React + TypeScript Web Metricas
Principalmente se adiciona aplicación en React con TypeScript para Metricas según archivo JSON data.json.

Decisiones Técnicas.
- Basicamente se optó por un desarrollo sencillo, práctico en react con typescript, utilizando de alguna forma
  la IA de Copilot, solamente para generar styles - y configuración en archivo app.tsx

2a Iteración.

- Fundamentalmente, en mi 2a iteración y cambios que habría que realizar, es darle más estilo a las pantallas,
  realizar traducción de los textos y glosas que se muestran en inglés, a varios idiomas, principalmente eso.
